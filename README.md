# Discord Datamining

이 레포지토리는 **Discord**의 클라이언트 및 API의 변경 사항을 자동으로 추적하고 기록하기 위해 생성되었습니다. 프로그램이 주기적으로 실행되어 실험(Experiments), 내부 문자열(Strings), API 엔드포인트 등의 변화를 감지하고, 변경 이력을 이 레포지토리에 자동으로 커밋 및 푸시합니다.

## 라이선스

이 프로젝트는 디스코드 아카이브 프로젝트의 일환으로 크리에이티브 커먼즈 라이선스(Creative Commons License) 저작자표시-동일조건변경허락(CC BY-SA)을 따릅니다.

적절한 저작자 표시를 하고, 2차 저작물을 동일한 라이선스 조건으로 배포하는 경우 공유 및 변경할 수 있습니다.
